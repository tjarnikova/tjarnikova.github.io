Colourmaps from old Czechoslovak Railways travel posters


I made this set of colourmaps from 1930s Czechoslovak Railways posters over a series of two rainy afternoons in October 2025. Posters were sourced from around the Internet, colours were extracted using Claude, followed by post-hoc manual tweaks. Code to make the example figures can be found here. I give the maps as a python list of hex codes, I use them in Python but of course they're usable anywhere you use a hex code. Free to use, acknowledgement appreciated.
